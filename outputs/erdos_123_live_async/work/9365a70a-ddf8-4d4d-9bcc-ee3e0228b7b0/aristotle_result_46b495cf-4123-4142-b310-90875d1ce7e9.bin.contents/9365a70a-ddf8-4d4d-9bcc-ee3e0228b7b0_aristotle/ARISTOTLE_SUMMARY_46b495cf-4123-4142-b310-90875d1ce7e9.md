# Summary of changes
## Summary

### What was accomplished

I worked on formalizing Erdős Problem 123 — the d-completeness of the triple-power semigroup `{a^i · b^j · c^k}` for pairwise coprime `a, b, c > 1`. This is a deep result in additive combinatorics.

**Proved lemmas** (11 fully verified, no sorry):
- **`dvd_iff_exponents_le`**: The key structural insight — divisibility between elements of PowTripleSet is equivalent to componentwise ≤ on exponent triples (i,j,k). This reduces the antichain problem from number theory to order theory on ℕ³.
- **`antichain_iff_exponents_incomparable`**: The antichain condition ↔ exponent incomparability.
- **`IsDComplete_mono`**: d-completeness is monotone under set inclusion.
- **`antichain_scale_a`**: Multiplying all elements by a generator preserves the antichain property.
- **`scale_mem_PowTripleSet[_b/_c]`**: Scaling preserves membership in PowTripleSet.
- **`represent_div_a`**: If `a | n` and `n/a` has an antichain representation, then `n` does too (via scaling). This is the key inductive step.
- Membership lemmas (`one_mem_PowTripleSet`, `pow_a_mem`, `pow_b_mem`, `pow_c_mem`, `mem_PowTripleSet`), closure (`PowTripleSet_mul_closed`), and positivity (`PowTripleSet_pos`).

**Remaining sorries** (2):
- **`residue_coverage`**: For every residue `r mod a`, find an antichain from `{b^j · c^k}` whose sum ≡ r mod a. This requires a pigeonhole argument on partial sums of anti-diagonal elements — a deep additive combinatorics result.
- **`erdos_123_d_complete_sequences`**: The main theorem, which follows from `residue_coverage` plus the scaling reduction.

### Key Discovery: Two generators can fail!

**The two-generator case is NOT always d-complete.** For `{8^i · 3^j}`, any antichain sum is ≡ 0, 1, or 3 mod 8 (since `3^j mod 8 ∈ {1,3}` and antichains in ℕ² use at most one row-0 element). Integers ≡ 2 mod 8 are never representable.

**Three generators overcome this** because "row 0" contains `{b^j · c^k}` with 2D antichains allowing multiple elements (e.g., `27 = 3³` and `15 = 3·5` are incomparable and sum to `42 ≡ 2 mod 8`).

### Which special cases transfer?

- **(2,3,5) and prime triples**: Transfer directly — powers of primes generate full multiplicative groups, so singleton coverage suffices.
- **(8,3,5) and composite generators**: Require the sumset mechanism — 2D antichains from the (j,k)-plane provide residue coverage via partial sums of anti-diagonal elements.
- The proof architecture (scaling reduction + residue coverage) is universal across all valid triples.
